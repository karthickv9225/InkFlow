InkFlow Blog Platform API

Version: 1.0.0

API is running. Use the following endpoints:

  • POST /api/auth/register - Register user
  • POST /api/auth/login - Login user
  • GET /api/public/feed - Get published blogs
  • GET /api/public/blogs/:slug - Get blog by slug
  • POST /api/blogs - Create blog (auth required)
  • GET /api/blogs - Get user's blogs (auth required)
  • PATCH /api/blogs/:id - Update blog (auth required)
  • DELETE /api/blogs/:id - Delete blog (auth required)
  • GET /api/health - Health check
